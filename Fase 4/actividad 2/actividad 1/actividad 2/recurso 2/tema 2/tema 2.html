CSS Grid

CSS Grid layout contiene funciones de diseño dirigidas a los desarrolladores de aplicaciones web. El CSS grid se puede
utilizar para lograr muchos diseños diferentes. También se destaca por permitir dividir una página en áreas o regiones principales,
por definir la relación en términos de tamaño, posición y capas entre partes de un control construido a partir de primitivas HTML.
